# Milk Management System (MERN)

A web application to manage milk inventory, suppliers, and sales. Built with MongoDB, Express.js, React, and Node.js.

## Features
- Manage milk products (CRUD)
- Manage suppliers (CRUD)
- Manage sales records (CRUD)
- Simple and modern React UI

## Project Structure
- `/client` - React frontend
- `/server` - Express.js backend
